# If, elif, else
----
Permet d'effectuer des control basé sur une condition

## C++
----
### if, else
----
```C++
if (condition){
	code;
}
else{
	code;
}
```

### Ternary
----
Un ternary est une expression compacte et simplifier d'un if else
```C++
<condition> ? <valeur si vrai> : <valeur si fausse>
```
