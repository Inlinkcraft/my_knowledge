# Variable
----
Contient une valeur et/ou information

## C++
----
### Déclaration de variable
----
```C++
// Type nom;
int test;

// Initiation de la variable
int test = 5;
```

### Constante
----
Variable qui peu juste être lut
```C++
cont int TEST = 2;
```
