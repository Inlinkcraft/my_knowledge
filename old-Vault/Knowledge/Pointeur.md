# Pointeur 
----
Un pointeur est une [[Variable|variable]] qui store l'adresse d'une autre [[Variable|variable]].

## Syntaxe
----
Pour déclaré un pointeur:
```C++
type* nom_de_la_variable;
type* nom_de_la_variable = &objet;
```
